\# Client Quote Generator



\## What this is

Python tool that generates professional PDF quotes for electrical jobs.

Built for a Canadian industrial electrician.



\## Built so far

\- quote\_generator.py — complete

\- Generates PDF quotes with materials, labour, overhead, grand total

\- PDF named quote\_\[clientname]\_\[date].pdf



\## Rules

\- Use Canadian Electrical Code (CEC) references where relevant

\- All dollar amounts in CAD

\- Use plain Python only — no unnecessary libraries

\- Comment every section of code

\- Explain what you built after building it

\- Default labor rate: $95/hr (adjust as needed)



\## Output

\- Generates a PDF quote ready to print or email to a client
